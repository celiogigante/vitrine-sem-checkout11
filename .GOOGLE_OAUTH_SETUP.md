# 🔐 Configuração Google OAuth - Guia Completo

## PASSO 1: Criar Projeto no Google Cloud

1. Acesse: https://console.cloud.google.com
2. Clique em **"Select a Project"** (topo esquerdo)
3. Clique em **"NEW PROJECT"**
4. Digite um nome: `Master Cell` (ou qualquer nome)
5. Clique **CREATE**
6. Aguarde o projeto ser criado (pode levar alguns segundos)

---

## PASSO 2: Habilitar Google+ API

1. Na barra de busca (topo), procure por: `Google+ API`
2. Clique em **"Google+ API"**
3. Clique no botão **ENABLE**
4. Aguarde a ativação

---

## PASSO 3: Configurar OAuth Consent Screen

1. No menu lateral, vá para: **APIs & Services** → **OAuth Consent Screen**
2. Selecione: **External**
3. Clique **CREATE**
4. Preencha assim:

   **App name:** `Master Cell`
   
   **User support email:** `celiommn@gmail.com`
   
   **Developer contact:** `celiommn@gmail.com`

5. Clique **SAVE AND CONTINUE**
6. Em **Scopes**, clique **SAVE AND CONTINUE** (deixe vazio)
7. Em **Test users**, clique **SAVE AND CONTINUE**
8. Revise e clique **BACK TO DASHBOARD**

---

## PASSO 4: Gerar Credenciais OAuth

1. No menu lateral: **APIs & Services** → **Credentials**
2. Clique em **+ CREATE CREDENTIALS** (botão azul, topo)
3. Selecione: **OAuth client ID**
4. Se pedir para configurar a tela de consentimento, clique **Go back**
5. Em **Application type**, selecione: **Web application**
6. Em **Name**, digite: `Supabase Master Cell`
7. Em **Authorized redirect URIs**, clique **ADD URI** e cole:
   ```
   https://uuwpzxpsvltqhrtadirk.supabase.co/auth/v1/callback
   ```
8. Clique **CREATE**
9. Uma janela vai aparecer com seus dados. **Copie e guarde:**
   - **Client ID** 
   - **Client Secret**

✅ Agora você tem as credenciais!

---

## PASSO 5: Configurar no Supabase

1. Abra seu painel Supabase: https://app.supabase.com
2. Vá em **Authentication** → **Providers** (menu esquerdo)
3. Procure por **Google**
4. Clique na engrenagem (⚙️) para editar
5. Cole seus dados:
   - **Client ID:** (copie do Google Cloud)
   - **Client Secret:** (copie do Google Cloud)
6. Clique **SAVE**

✅ Google OAuth está configurado!

---

## PASSO 6: Testar o Painel Admin

1. Acesse seu site: http://localhost:8080/admin/login (ou a URL do seu site)
2. Clique em **"Entrar com Google"**
3. Faça login com `celiommn@gmail.com`
4. Você deve ser redirecionado para `/admin` com sucesso!

---

Seu painel admin agora tem autenticação segura com Google OAuth!
